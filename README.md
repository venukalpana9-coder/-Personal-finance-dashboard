Personal Finance Dashboard
Project Description
The Personal Finance Dashboard is a Python project that helps users manage their daily finances. It allows users to add income and expenses, view their current balance, and keep a record of all transactions. The data is stored in a JSON file so it is available even after closing the program.
Features
Add income
Add expenses
View current balance
View transaction history
Save data in JSON format
Simple command-line interface
Technologies Used
Python 3
JSON
Project Structure
personal-finance-dashboard/
│── main.py

│── data.json
│── README.md
└── requirements.txt
How to Run
Clone the repository:
git clone https://github.com/your-username/personal-finance-dashboard.git
Open the project folder:
cd personal-finance-dashboard
Run the project:
python main.py
